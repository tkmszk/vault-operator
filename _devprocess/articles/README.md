# articles/ - Externe Inhalte

Sammelpunkt fuer alle Inhalte, die fuer eine **externe Audience**
geschrieben sind: Blog-Posts, Tutorial-Drafts, Reddit-Posts, LinkedIn,
Medium.

Diese Inhalte sind nicht Teil des V-Model-Workflows und folgen
keiner Praefix-Konvention.

## Konvention

- Slug-basierte Filenames: `artikel-token-reduktion-medium.md`
- Sprache: Frei (DE oder EN), je nach Zielmedium
- Status: nicht in Backlog gepflegt

## Was hier NICHT hingehoert

- Interne Doku -> `_devprocess/analysis/RESEARCH-NNN-`
- Public-Repo-Doku (README, docs/) -> Repo-Root, nicht hier
- Marketing-Material das spaeter public werden soll -> Repo-Root `docs/`
